# 联调与验收清单

## 范围

本清单用于阶段 9，覆盖：

- 企业基础信息
- 登录与企业隔离
- 单据、附件与分析生成
- 台账与税务配置
- 勾稽校验
- 打印与 PDF 导出

## 核心验收路径

### A. 企业与权限基础

- 登录成功后只能看到本企业数据
- 企业基本信息修改后，单据、报表、税务预览同步更新
- 不同角色访问受限接口时返回正确状态码

### B. 单据与附件

- 录入行为分析成功后可按锁定结果生成单据
- 单据新增后可在总库查询到
- 附件补传后能绑定到目标单据
- 单据详情和附件状态一致

### C. 台账与报表

- 凭证列表可查
- 总账、明细账、科目余额表可查
- 银行日记账、现金日记账可查
- 三大财务报表可按月/季/年切换

### D. 纳税人口径

- 一般纳税人路径可查
- 小规模纳税人路径可查
- 简易计税路径可查
- 税务事项查询与企业纳税人配置一致

### E. 勾稽校验

- 合同、开票、回款、收入确认勾稽可触发
- 工资、个税、社保、公积金勾稽可触发
- 研发项目、辅助账、汇算优惠勾稽可触发
- 异常问题可返回严重级别和说明

### F. 打印与导出

- 单据打印任务可创建
- PDF 导出任务可创建
- 归档目录导出任务可创建
- 税务底稿导出任务可创建

## 最低通过标准

- 所有核心接口返回结构稳定
- 关键页面可读写企业主数据
- 单据、台账、税务、勾稽、打印五条主链路至少各自打通一条最小闭环
